# App Compatibility with GrapheneOS
Just a collection of apps I've tested that work without Google Play Services on GrapheneOS. Feel free to add apps that you tested to the list. List can be found under the 'Code' section of this repo
# Contributing
You can help make this list better for everyone by adding apps you've tested on GrapheneOS that work properly without Google Play Services. Just include the app name, a link to the app, and specify if any special settings (eg; Exploit Protection Compatibility Mode) need to be turned on. Please read the CONTRIBUTING.md file for more details
